# GHSL-2024-070: Chromium中的远程代码执行漏洞 - CVE-2024-3833

在受影响的Chrome版本中，访问恶意网站可能导致对象损坏，从而让攻击者在Chrome的渲染器中执行代码。

## 关键技术点

- **受影响的版本**: Chromium 版本 123.0.6312.58
- **漏洞类型**: 远程代码执行（RCE）

### 漏洞细节

1. **WebAssembly对象问题**:
   - 在某个函数`InstallConditionalFeatures`中，有一个检查确保WebAssembly对象不包含`Function`属性，以避免重复。
   - 但在另一个函数`InstallTypeReflection`中，对其他对象的属性没有进行类似检查。

2. **结果**:
   - 这导致某些对象如`wasm_table_constructor`的原型被添加新属性，而不检查这些属性是否已经存在，进而引发重复属性的问题。

3. **利用漏洞**:
   - 攻击者可以利用这个问题在Chrome渲染器中执行恶意代码。
