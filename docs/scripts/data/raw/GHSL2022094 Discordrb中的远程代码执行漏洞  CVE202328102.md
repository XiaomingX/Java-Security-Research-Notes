# GHSL-2022-094: Discordrb中的远程代码执行漏洞 - CVE-2023-28102

## 概述

在 `discordrb` 库中，`encode_file` 方法可能会导致远程代码执行风险，特别是当它使用不可信的用户输入时。

### 版本

- 最新版本: 3.4.1

## 漏洞详情

在 `encoder.rb` 文件中，`encode_file` 方法使用文件参数构建了一个不安全的 shell 命令字符串。这可能导致使用 `discordrb` 的客户端受到命令注入攻击的风险。

### 利用条件

- 该库本身并不可直接被利用：攻击者需要确保某个使用该库的客户端调用了有漏洞的方法，并传入用户输入。
- 如果不安全的输入被传递到库的方法中，攻击者就有可能在主机上执行任意的 shell 命令。

### 影响

如果漏洞被成功利用，攻击者可以在运行该库的机器上执行恶意代码，从而可能造成 **远程命令执行 (RCE)**。

### 漏洞示例

- 发现此漏洞的代码分析结果是 CodeQL。
- 一个概念验证（PoC）示例显示，漏洞会在当前工作目录下创建一个名为 `pwned` 的新文件。
