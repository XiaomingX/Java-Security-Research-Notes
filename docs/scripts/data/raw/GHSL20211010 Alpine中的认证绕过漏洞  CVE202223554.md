# GHSL-2021-1010: Alpine中的认证绕过漏洞 - CVE-2022-23554

## 概述
在Alpine 1.10.2版本中，存在一个认证绕过的漏洞。

## 技术细节
- **问题描述**: 这个漏洞允许攻击者绕过AuthenticationFilter（认证过滤器）。
- **请求URI**: 认证过滤器通过检查请求的URI来决定用户是否可以访问Swagger接口。
- **漏洞利用**: 如果访问的URL路径格式为`/api/foo;%2fapi%2fswagger`，认证过滤器会认为条件满足，随即返回结果，而不终止请求处理。
- **影响**: 虽然会绕过认证，但攻击者无法获得用户身份（principal对象不会被分配），因此不会导致用户冒充问题。
