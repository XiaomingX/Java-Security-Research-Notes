# GHSL-2023-257: Plane中的服务器端请求伪造（SSRF）漏洞 - CVE-2024-31461

## 概述
2024年3月11日：发送跟进邮件询问修复的新版本何时发布。

Plane v0.13-dev 存在身份验证盲的服务器端请求伪造（SSRF）漏洞。

## 漏洞详情
- **漏洞影响版本**：v0.13-dev
- **发现位置**：apiserver/plane/api/views/importer.py 文件中的 ServiceIssueImportSummaryEndpoint 类的 `get` 函数。
- **漏洞原因**：该函数接受一个用户控制的参数 `cloud_hostname`，并将其不安全地与其他 URL 拼接后用于向任意目标发送 GET 请求。

### 影响
- 此漏洞可能导致信息泄露。攻击者可以利用该漏洞，通过服务器向任意地址发送 GET 请求。

## 漏洞发现
- 本漏洞是通过 CodeQL 工具发现的。
