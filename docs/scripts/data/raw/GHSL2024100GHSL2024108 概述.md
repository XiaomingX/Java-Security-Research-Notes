# GHSL-2024-100_GHSL-2024-108 概述

## 漏洞简介
在 **Streamlit-geospatial** 项目中发现了多个**远程代码执行（RCE）**和**盲服务器端请求伪造（SSRF）**漏洞。这些漏洞可能会导致严重的安全风险。

## 关键漏洞说明

### 远程代码执行（RCE）
远程代码执行允许攻击者在服务器上执行任意代码，以下代码片段中的变量接收用户输入并被传递给 `eval()` 函数：

1. **`palette` 变量（文件: pages/1_📷_Timelapse.py）**
   - 行 380
   - 行 430
   - 行 488

2. **`vis_params` 变量（文件: pages/1_📷_Timelapse.py）**
   - 行 383
   - 行 390
   - 行 1254

3. **`vis_params` 变量（文件: pages/10_🌍_Earth_Engine_Datasets.py）**
   - 行 115

4. **`vis_params` 变量（文件: 8_🏜️_Raster_Data_Visualization.py）**
   - 行 80

### 盲服务器端请求伪造（SSRF）
盲 SSRF 允许攻击者通过服务器发送请求到内部网络，以下代码片段中用户输入的 URL 变量用于构造请求：

1. **第一处（文件: pages/1_📷_Timelapse.py）**
   - 行 47：传递给 `get_layers` 函数，可能导致伪造请求。

2. **第二处（文件: pages/1_📷_Timelapse.py）**
   - 行 63：传递给 `gpd.read_file` 方法，可能导致伪造请求。

## 安全影响
- **RCE** 可能导致攻击者在服务器上执行任意代码。
- **盲 SSRF** 允许攻击者探测服务器及其内部网络的其他漏洞。
