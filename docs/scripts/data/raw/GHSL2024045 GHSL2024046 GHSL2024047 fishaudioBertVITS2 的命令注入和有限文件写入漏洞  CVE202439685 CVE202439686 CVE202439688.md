# GHSL-2024-045, GHSL-2024-046, GHSL-2024-047: fishaudio/Bert-VITS2 的命令注入和有限文件写入漏洞 - CVE-2024-39685, CVE-2024-39686, CVE-2024-39688

## 概述
fishaudio/Bert-VITS2 版本 2.3 存在命令注入和有限文件写入的安全漏洞。

## 技术细节

### 1. 命令注入
- **漏洞位置**：
  - `resample` 函数中的 `subprocess.run(cmd, shell=True)` （行 46-52）
  - `bert_gen` 函数中的 `subprocess.run(cmd, shell=True)` （行 82）
  
- **原因**：用户输入的 `data_dir` 变量被直接用于构建命令。这允许攻击者在服务器上执行任意命令。

### 2. 有限文件写入
- **漏洞位置**：
  - `generate_config` 函数 （行 34）
  
- **原因**：用户输入的 `data_dir` 变量与其他文件夹路径拼接后用于打开新文件。攻击者可以在错误的目录下写入 `/config/config.json` 文件。虽然如果目录不存在，应用会返回错误，但这可以让攻击者获取服务器上存在的目录信息。
