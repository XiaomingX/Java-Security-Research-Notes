# GHSL-2024-127_GHSL-2024-129：OpenC3 COSMOS中的远程代码执行（RCE）漏洞

在OpenC3 COSMOS这个用于控制卫星和测试设备的Web应用程序中发现了多个漏洞。这些漏洞可能导致远程代码执行（RCE）和跨站脚本攻击（XSS）。

## 漏洞概述

- **版本**: OpenC3 COSMOS v5.16.2
- **相关CVE**:
  - CVE-2024-43795: 路径遍历漏洞
  - CVE-2024-46977: 反射式跨站脚本漏洞
  - CVE-2024-47529: 明文存储敏感信息漏洞

### 1. 路径遍历漏洞 (CVE-2024-43795)

- **描述**: 认证用户可以通过不当路径遍历来下载服务器上的任意`.txt`文件。
- **原理**:
  - 用户控制的参数`target`和`screen`经过处理形成文件路径。
  - 如果环境变量`OPENC3_LOCAL_MODE`被设置为1，则有可能利用这一漏洞。
  
- **示例**: 
  - 要下载`/openc3/python/requirements.txt`文件，需要构造特定的URL请求。

### 2. 反射式跨站脚本漏洞 (CVE-2024-46977)

- **描述**: 在登录页面中，攻击者可以利用恶意的重定向参数执行伪造JavaScript代码。
- **原理**:
  - 通过将恶意脚本作为重定向参数提供，攻击者可以在用户登录后执行任意JavaScript代码。

- **示例**: 
  - 攻击者可以发送特定链接，成功使用户输入密码并触发恶意脚本。

### 3. 明文存储敏感信息漏洞 (CVE-2024-47529)

- **描述**: 用户的密码以明文形式存储在浏览器的LocalStorage中，可能被恶意代码提取。
- **原理**:
  - 如果攻击者利用XSS漏洞，能轻松提取用户密码或令牌，并可能将其发送到控制的远程服务器。

## 结论

这些漏洞可能导致信息泄露或远程代码执行，影响OpenC3 COSMOS的安全性。
