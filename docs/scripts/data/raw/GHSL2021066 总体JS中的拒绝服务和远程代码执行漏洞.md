# GHSL-2021-066: 总体JS中的拒绝服务和远程代码执行漏洞

## 总体JS简介
- **版本**: 最新版本为 3.4.8（可通过 npm 获取）。

## 漏洞描述
- **问题**: 通过调用 `utils.set` 函数并传入用户控制的值，可以导致代码注入。
- **影响**: 攻击者能在 Node.js 环境中执行任意的 JavaScript 代码。

## 漏洞证明
### 1. 拒绝服务（DoS）
- **原理**: 该漏洞可以通过使程序进入无限循环来造成服务不可用。

### 2. 代码执行（RCE）
- **原理**: 该漏洞能够创建一个名为 `GHSL` 的文件在当前工作目录中。
