# GHSL-2024-030: 潜在的秘密泄露风险

## 概述

在docfx仓库中，不安全地使用`pull_request_target`导致了秘密（如API密钥）的泄露风险。

- **受影响的项目**: dotnet/docfx
- **版本**: 2.75.3

## 问题描述

使用GitHub的`pull_request_target`事件时，会直接检出（checkout）Pull Request的源代码，这可能使得来自攻击者的代码被执行。

因为这个不可信的代码在可以访问秘密的环境中运行，攻击者有可能提交一个含有恶意`templates/packages.json`文件的Pull Request，导致执行任意命令并获取与工作流相关的秘密（例如：`secrets.PERCY_TOKEN`）。

## 漏洞验证

该漏洞是通过CodeQL查询“在可信环境中检出不受信任代码”发现的。

### 风险

运行不可信的代码，并使用带有权限的仓库令牌和访问秘密的能力，可能导致未经授权的仓库修改或秘密泄露。
