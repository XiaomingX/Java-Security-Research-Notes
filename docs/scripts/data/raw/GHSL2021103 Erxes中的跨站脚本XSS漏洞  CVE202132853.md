# GHSL-2021-103: Erxes中的跨站脚本（XSS）漏洞 - CVE-2021-32853

## 什么是跨站脚本（XSS）？

跨站脚本（XSS）是一种安全漏洞，攻击者可以在某个网页中插入恶意代码。当用户访问该网页时，恶意代码会在他们的浏览器中执行，可能导致信息泄露或其他恶意行为。

## 漏洞描述

在Erxes（https://github.com/erxes/erxes）的某些小部件（widgets.ejs）中，存在一个可以执行代码的漏洞。具体情况如下：

1. **漏洞位置**：某个模板标签受到影响；
2. **输入来源**：用户的请求参数；
3. **缺乏安全处理**：插入的值没有经过转义，攻击者可以将恶意代码插入到字符串中。

用户访问包含恶意链接的网页时，可能导致客户端代码执行。
