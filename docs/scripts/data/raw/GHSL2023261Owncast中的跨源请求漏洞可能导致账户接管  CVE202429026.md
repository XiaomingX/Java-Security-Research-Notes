# GHSL-2023-261：Owncast中的跨源请求漏洞可能导致账户接管 - CVE-2024-29026

## 关键技术点

- **CORS（跨源资源共享）**：是一种安全功能，允许网页从不同的域请求资源。若CORS策略设置宽松，会导致安全隐患。

- **漏洞描述**：
  - 在Owncast（版本0.1.2）中，CORS策略过于宽松，使得攻击者能够发送跨域请求，并读取敏感信息。
  - 具体来说，攻击者可以通过一个恶意网站，获取到管理员的密码。

- **攻击方式**：
  - 攻击者可以在其网站上托管一个特定的HTML文件。
  - 当一个已登录的Owncast管理员访问该网站时，恶意HTML会窃取管理员密码并进行提示（弹出警告）。

- **研究与发现**：
  - 该漏洞是通过CodeQL的CORS查询工具发现的。
  - 发现者：GHSL团队成员 @Kwstubbs（Kevin Stubbings）。
