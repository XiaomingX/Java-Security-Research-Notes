# GHSL-2023-004: 任意文件上传与下载漏洞 - CVE-2023-22726

## 漏洞简介
GitHub Action的工件服务器存在一个安全漏洞，导致攻击者可以通过特定操作下载和覆盖服务器上的任意文件。

## 受影响版本
- **act**: v0.2.35

## 漏洞细节
1. **上传漏洞**: 
   - `/upload` 接口存在路径遍历漏洞。用户可以控制文件路径，最终这些路径会传递给 `os.Mkdir` 和 `os.Open` 函数。

2. **下载漏洞**: 
   - `/artifact` 接口同样存在路径遍历漏洞。用户控制的路径将决定服务器返回的文件。

### 风险
- **特权提升**: 攻击者可能利用此漏洞获得更高的权限。
