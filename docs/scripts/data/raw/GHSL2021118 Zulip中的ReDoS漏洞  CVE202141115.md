# GHSL-2021-118: Zulip中的ReDoS漏洞 - CVE-2021-41115

## 什么是ReDoS？

- **ReDoS**（正则表达式拒绝服务）是一种安全漏洞。
- 发生在使用效率低下的正则表达式时，这种正则表达式在处理特制输入字符串时会非常缓慢，可能导致服务中断。

## 漏洞概述

- **产品**: Zulip 4.5
- **发现方法**: 使用CodeQL查询来识别不高效的正则表达式。
- **漏洞数量**: 有两个正则表达式存在此漏洞。

## 重现步骤

1. 在一个实际的Zulip组织中重现该问题。
2. 或者，可以将漏洞的正则表达式复制粘贴到一个独立文件中，以检测其脆弱性。

## 影响

- 该漏洞可能导致拒绝服务（Denial of Service）。
