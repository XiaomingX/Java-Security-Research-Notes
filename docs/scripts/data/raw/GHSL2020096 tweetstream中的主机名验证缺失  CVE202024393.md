# GHSL-2020-096: tweetstream中的主机名验证缺失 - CVE-2020-24393

## 概述

在`tweetstream`库的版本`v2.6.1`中，缺少主机名验证会导致中间人攻击（Man-in-the-Middle，MITM）。攻击者可以假冒可信服务器，向用户发送恶意数据。

## 关键技术点

- **中间人攻击**：攻击者能够在用户和可信服务器之间插入自己，从而窃取或篡改数据。
  
- **事件驱动库**：`tweetstream`使用`eventmachine`库，但未安全使用，导致主机名验证失效。

- **触发条件**：只需一个简单的TLS支持的监听程序，就可以利用这个漏洞。

## 漏洞编号

- **CWE-297**：证书主机名不匹配的验证不当。
  
- **CVE-2020-24393**：该漏洞的编号。
