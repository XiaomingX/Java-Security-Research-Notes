# GHSL-2024-089: youtube-dl中的路径遍历漏洞，导致远程代码执行 - CVE-2024-38519

## 概述
youtube-dl在处理字幕文件扩展名时没有进行验证，导致Windows用户面临路径遍历攻击，攻击者可以通过特制的链接在下载视频时覆盖任意二进制文件。

## youtube-dl基本信息
- 受影响版本：nightly v2024.05.16 和 最后一个正式版本 v2021.12.17。
- 功能：youtube-dl不仅可以下载视频文件，还可以下载字幕，用户可以通过以下选项来实现：`--write-sub`, `--write-auto-sub`, 或 `--all-subs`。

## 漏洞细节
- **字幕文件提取**：支持的视频网站有专门的提取器，有些提取器验证字幕扩展名，而有些则将提取的值直接信任。
- **输出文件命名**：youtube-dl使用给出的扩展名生成输出文件名，并且在Windows平台上，路径遍历攻击生效。
- **漏洞造成的影响**：如果攻击者构造了恶意的字幕扩展名，用户下载视频时可能会不小心覆盖重要文件，最终可能导致远程代码执行。

## 漏洞利用
- 攻击者可以通过XSS或SQL注入等方式利用特定的youtube-dl提取器。如果用户下载了含有恶意字幕的文件，可能导致未授权的文件系统修改和远程代码执行。
