# GHSL-2020-258: ZipSlip漏洞在bblfshd中 - CVE-2021-32825

## 概述
ZipSlip漏洞是一种安全隐患，允许攻击者通过不当处理符号链接，在解压缩一个包含恶意内容的压缩包时，读取或写入指定目录之外的任意位置。

## 技术原理
- **符号链接**：一种特殊的文件类型，指向另一个文件或目录。
  
- **解压缩流程**：
  - 在解压缩tar文件时，程序会检查并防止创建指向外部目录的符号链接。
  - 然而，攻击者可以创建一个特殊格式的tar包，使得解压缩时的符号链接（例如链接到上级目录）绕过这一检测。

### 示例
1. **创建攻击性tar包**：
   - 在一个子目录中，创建指向父目录的链接。
   - 该链接可以让攻击者访问到不被允许的文件。
   
2. **后果**：
   - 如果攻击者可以控制压缩包，则可能允许任意文件写入，权限与执行解压操作的程序相同。
   - 此外，攻击者可读取解压文件，如果原程序具有对其他系统文件的读取权限，攻击者也可能获取这些文件。
