# GHSL-2024-071: Chromium中的内存损坏 - CVE-2024-3832

在受影响版本的Chrome中，打开恶意网站可能导致Chrome渲染器中的对象被破坏。

## 涉及的版本
- Chromium版本：123.0.6312.58

## 问题描述
之前为防止类似问题（如CVE-2021-30561）而引入的机制不够有效。在"InstallConditionalFeatures"中，增加了一项检查，确保在将WebAssembly对象添加到对象中时，不具有相关属性。

### 检查机制
- 在检查中使用的对象是全局的WebAssembly属性。
- 但在安装过程中，使用的却是上下文中的WebAssembly对象，这可能与全局WebAssembly属性不一致。

如果先在WebAssembly对象上设置"Suspender"属性，然后将全局WebAssembly设置为不同的对象，则会导致属性名称检查作用于新分配的WebAssembly对象，而属性却添加在上下文中已有的WebAssembly对象上。这会导致一个被破坏的对象，产生重复的Suspender属性。此外，这个问题还影响了"InstallTypeReflection"函数的行为。

## 影响
该问题可能会导致Chrome渲染器中的内存损坏。
