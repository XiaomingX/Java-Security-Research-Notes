# GHSL-2020-254: dotmesh中的任意文件读取和/或写入 - CVE-2020-26312

## 问题简介
dotmesh在解压文件时对符号链接的处理不安全，可能允许攻击者在指定目标文件夹外部读取和/或写入任意位置的文件。

## 技术细节
- **影响的版本**: 截至报告时间（2020年11月27日）的最新提交。
- **问题代码**: `untarFile`函数试图防止创建指向解压目录外的符号链接，但存在漏洞。
- **漏洞利用**: 如果恶意的tar文件首先链接子目录到上级目录（..），然后再链接另一子目录到上级目录，这会导致符号链接指向tar文件的父目录，从而绕过安全检查。

## 潜在影响
- **任意文件写入**: 攻击者可以控制归档文件，从而在运行解压操作的程序相同权限下写入文件。
- **任意文件读取**: 如果攻击者能读取解压后的文件，可能会读取到父进程有权限访问的任意系统文件。
