# GHSL-2023-250: DocsGPT 的未认证文件写入漏洞 - CVE-2024-31451

### 概述
DocsGPT 存在一个未认证的有限文件写入漏洞。

### 受影响版本
- DocsGPT v0.5.0

### 漏洞描述
- `check_docs` 接口接收用户发送的 POST 数据，并将其添加到一个 URL 中。
- 这种方式使得攻击者可以改变下载文件的来源 URL。
- 受影响的文件被保存在服务器上。

### 检测工具
- 该漏洞是通过使用 CodeQL 工具发现的。

### 漏洞影响
- 此漏洞可能导致有限文件写入。
