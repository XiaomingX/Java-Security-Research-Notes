# GHSL-2022-048: JSONXX库的双重释放漏洞 - CVE-2022-23459

## 漏洞概述
在 JSONXX（版本 1.0.1）库中，`Value` 类存在双重释放（Double Free）或使用已释放内存（Use After Free）的问题。这主要是因为 `Value` 类使用了默认的赋值运算符，但它包含指针成员。当特定情况下使用时，可能导致内存损坏。

## 漏洞影响
攻击者可以利用这个漏洞执行任意代码，造成安全风险。

## 漏洞发现
该问题由 GHSL 团队成员 @JarLob（Jaroslav Lobačevski）发现并报告。
