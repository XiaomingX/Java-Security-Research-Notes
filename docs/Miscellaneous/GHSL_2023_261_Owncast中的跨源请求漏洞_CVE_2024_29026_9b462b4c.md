# GHSL-2023-261: Owncast中的跨源请求漏洞（CVE-2024-29026）

## 概述
Owncast（版本0.1.2）存在一个跨源资源共享（CORS）政策宽松的问题，允许攻击者发起跨源请求并读取敏感信息，例如管理员密码。这可能导致帐户被接管。

## 技术细节
- **CORS（跨源资源共享）**：是一种允许或限制浏览器从不同源（域）加载资源的机制。
- **漏洞描述**：Owncast的CORS处理程序会反射Origin头，这意味着如果攻击者提供一个包含恶意代码的网站，已经登录的管理员访问该网站时，攻击者可以窃取他们的密码。
  
## 攻击方式
攻击者可以在他们的网站上托管以下HTML文件。当已登录的Owncast管理员访问这个网站时，管理员的密码会被盗取，并显示在浏览器的警告框中。
