# GHSL-2020-096: 缺失主机名验证（CVE-2020-24393）

## 概述
在`tweetstream`库中，缺少对主机名的验证，这使得攻击者能够进行中间人攻击。

## 受影响的版本
- `tweetstream`版本：**2.6.1**

## 问题描述
`tweetstream`库不安全地使用了`eventmachine`，这使得攻击者可以伪装成一个可信任的服务器，向用户发送恶意数据。

### 漏洞触发
攻击者只需设置一个简单的支持TLS的监听程序，就可以利用这一漏洞。以下是操作步骤的简要说明：

1. 创建一个样例客户端。
2. 运行该客户端，观察到连接建立到之前初始化的监听程序。

## 漏洞类型
- **CWE-297**: 不当验证证书导致主机不匹配
