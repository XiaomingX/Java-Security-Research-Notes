# GHSL-2022-061: Bearer Token 泄露问题 - CVE-2022-39304

## 概述
- **发布日期**: 2022年12月19日
- **问题描述**: 在进行令牌更新时，如果发生错误，应用的完整 Bearer 令牌可能会被泄露。这可能会出现在 Slack 频道或类似应用中。

## 技术细节
- **受影响版本**: ghinstallation 版本最高至 2.1.0
- **泄露过程**: 当令牌更新出现错误时，系统会输出应用的完整 Bearer 令牌。这有可能导致令牌被外部获取，进而导致应用被劫持。
