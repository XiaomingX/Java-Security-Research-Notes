# GHSL-2024-096: DNS重绑定漏洞 - CVE-2024-42364

## 概述
Homepage 在默认设置下存在 DNS 重绑定漏洞，攻击者可以借此访问主页所有者的私密信息。

## 漏洞详情
- **软件版本**: homepage v0.9.1
- **漏洞类型**: DNS 重绑定

## 技术背景
1. **DNS 重绑定**:
   - 攻击者会诱使用户访问其网站。
   - 在用户访问时，攻击者将其域名的 DNS 记录从自己的 IP 地址更改为主页实例的内部 IP 地址。
   - 通过绑定子域名到不同的 IP 地址，攻击者可以检查哪些 IP 地址有效。

2. **攻击过程**:
   - 一旦找到有效的内部 IP 地址，攻击者可以读取来自主页实例的响应。
   - 攻击者网站的请求会被主页实例处理，因此可以获取用户的私密信息，例如 API 密钥等。

## 风险
- **信息泄露**: 攻击者能够从与受害者在同一内网的 homepage 实例中窃取信息。
