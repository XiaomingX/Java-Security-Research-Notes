# GHSL-2021-103: Erxes中的跨站脚本攻击（XSS）问题 - CVE-2021-32853

### 什么是跨站脚本攻击（XSS）？
跨站脚本攻击（XSS）是一种网络安全漏洞，攻击者可以在网页中插入恶意代码，当用户访问该网页时，恶意代码会在用户的浏览器中执行。

### 问题概述
- **受影响项目**: [Erxes](https://github.com/erxes/erxes)
- **最新版本**: 截至目前的最新版本

### 技术细节
- **漏洞位置**: 在文件 `widgets.ejs` 中的一个模板标签存在代码注入的漏洞。
- **漏洞原因**: 从请求参数获取的值没有进行适当的转义处理，这使得攻击者可以插入恶意代码，打破字符串文字或脚本标签的封闭。

### 风险
- **执行的代码**: 恶意代码将在客户端执行。
- **攻击途径**: 受害者需要点击恶意链接或从恶意网站重定向到受影响的网站。
