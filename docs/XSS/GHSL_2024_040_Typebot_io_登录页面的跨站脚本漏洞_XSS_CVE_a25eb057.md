# GHSL-2024-040: Typebot.io 登录页面的跨站脚本漏洞 (XSS) - CVE-2024-30264

## 概述
在 typebot.io 的登录页面存在一个反射型跨站脚本漏洞 (XSS)，攻击者可以利用这个漏洞劫持用户账户。

## 技术细节
- **影响版本**: typebot.io v2.23.0
- **漏洞原理**:
  - 登录页面从 URL 中获取 `redirectPath` 参数。
  - 如果用户点击了一个包含恶意 `redirectPath` 的链接，并且这个参数使用了 JavaScript 方案，攻击者可能会执行任意 JavaScript 代码，具有用户的权限。

## 潜在风险
这个漏洞可能导致账户被劫持。

攻击者可以构造如下链接，从本地服务器下载并在受害者的域名下执行某个 JavaScript 文件。这个方法允许攻击者在不知情的情况下执行其代码。
