# GHSL-2021-104: Countly Server中的跨站脚本漏洞 - CVE-2021-32852

## 描述

在 [Countly Server](https://github.com/Countly/countly-server) 中发现了一个跨站脚本（XSS）漏洞。

## 漏洞详情

- 漏洞位置：`reset.html` 文件中的模板片段。
- 影响：攻击者可以通过设置`message`和`password_min`参数进行代码注入。
  
### 代码注入示例

```html
<%= message %>
```

- 该标签通常会对HTML进行转义，所以不允许跳出字符串。
- 但是，反斜杠没有被转义，攻击者可以利用这一点进行注入。

### 攻击方式

1. 受害者需要点击恶意链接或从恶意网站重定向。
2. 攻击者必须拥有一个账户或能够创建一个账户。

## 版本影响

- 目前尚不清楚该开源代码是否与企业版软件相关，可能只有社区版本受影响。
