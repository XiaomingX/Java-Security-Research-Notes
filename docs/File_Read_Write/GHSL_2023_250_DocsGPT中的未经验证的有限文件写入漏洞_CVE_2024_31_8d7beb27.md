# GHSL-2023-250：DocsGPT中的未经验证的有限文件写入漏洞 - CVE-2024-31451

## 概述
DocsGPT（版本0.5.0）存在一个未经验证的文件写入漏洞，攻击者可以利用此漏洞进行有限的文件写入。

## 漏洞详情
- **漏洞位置**：`check_docs` 接口。
- **漏洞原因**：该接口将用户提交的POST数据与一个URL连接起来，这使得攻击者可以篡改下载文件的源库。
- **影响**：恶意用户可能能够在服务器上写入有限的文件。

## 漏洞发现
- 该漏洞是通过 **CodeQL** 工具发现的。
- 发现并报告该问题的人员是GHSL团队成员**Sylwia Budzynska**。
