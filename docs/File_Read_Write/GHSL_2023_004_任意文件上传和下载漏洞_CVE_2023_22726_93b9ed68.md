# GHSL-2023-004: 任意文件上传和下载漏洞 - CVE-2023-22726

## 漏洞概述
GitHub Action 的 artifacts 服务器没有对路径输入进行处理，这使得攻击者能够在 GitHub Action 中下载和覆盖主机上的任意文件。

## 受影响的版本
- **act**: v0.2.35

## 漏洞细节
1. **上传漏洞**：  
   - `/upload` 接口存在路径遍历漏洞，用户可以控制文件路径，最终导致调用 `os.Mkdir` 和 `os.Open` 操作。

2. **下载漏洞**：  
   - `/artifact` 接口同样存在路径遍历漏洞，用户可以控制路径变量，服务器将返回指定的文件。

这些漏洞可能导致特权升级。
