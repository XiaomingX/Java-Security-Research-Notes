# GHSL-2023-277: Owncast中的任意文件删除漏洞 (CVE-2024-31450)

## 概述
Owncast版本0.1.2存在一个漏洞，允许拥有管理员权限的远程攻击者通过发送恶意POST请求到 `/api/admin/emoji/delete` 来删除任意文件。

## 技术细节
- **应用程序**: Owncast
- **版本**: 0.1.2
- **API端点**: `/api/admin/emoji/delete`
  
该API允许管理员删除保存于磁盘上的自定义emoji。请求中包含的参数会被直接用于构建文件路径，删除指定的emoji。

### 漏洞关键点
- 攻击者可以利用"路径遍历"（例如使用`../`）的技术。
- 这使得攻击者可以删除系统中emoji目录以外的任意文件，造成严重的安全隐患。
