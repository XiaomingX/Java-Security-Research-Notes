# GHSL-2023-250: DocsGPT 的文件写入漏洞 - CVE-2024-31451

## 简介
DocsGPT（版本 v0.5.0）存在一个未经身份验证的有限文件写入漏洞。

## 漏洞描述
- **漏洞类型**：未经身份验证的有限文件写入
- **影响**：攻击者可以通过特定方式修改文件下载路径，导致服务器上文件被不当写入。

## 具体技术点
- **问题所在**：在 `check_docs` 接口中，用户提交的 POST 数据会与一个 URL 合并，允许攻击者利用此功能改变文件的来源。
- **代码行数**：相关代码在第 214 和 219 行。

## 漏洞发现
- **工具**：漏洞是通过 CodeQL 工具发现的。
- **报告人**：该问题由 GHSL 团队成员 @sylwia-budzynska（Sylwia Budzynska）发现并报告。
