# GHSL-2024-090: yt-dlp中的路径遍历漏洞 - CVE-2024-38519

`yt-dlp` 是一个可以下载视频及其字幕的工具。但是，它在处理字幕文件扩展名时没有进行验证，这使得Windows用户容易受到路径遍历攻击，攻击者可以通过恶意链接在下载时覆盖任意二进制文件。

## 技术基础知识

### yt-dlp功能
- **下载视频和字幕**：使用选项 `--write-subs`, `--write-auto-subs`, `--all-subs` 或 `--write-srt`。
- **提取器**：yt-dlp使用不同的网站提取器来获取视频和字幕链接。部分提取器可以直接提供字幕的URL和扩展名。

### 路径遍历漏洞
- 当用户下载视频及字幕时，yt-dlp 将根据提取的文件扩展名生成字幕的输出文件名。文件名格式为 `%(title)s [%(id)s].%(sub_lang).%(sub_format)s`。
- **路径遍历**：在Windows系统上，一些错误的字幕扩展名可能导致路径遍历攻击，使攻击者能够覆盖本地文件。Linux系统不受此影响，因为它们不允许某些类型的路径。

### 潜在风险
- 如果攻击者能够提供恶意的字幕扩展名，用户在下载时可能会无意中覆盖本地的重要文件。
- 这可能导致未授权的文件系统修改，并进一步可能导致远程代码执行（RCE）。
