# GHSL-2020-258: ZipSlip 漏洞在 bblfshd 中 (CVE-2021-32825)

## 概述
bblfshd 中存在一个漏洞，允许攻击者通过不安全地处理符号链接，读取或写入指定目标文件夹之外的位置。

## 漏洞细节
- **问题描述**: 在解压缩 tar 文件时，程序尝试防止创建指向解压目录外的符号链接。然而，攻击者可以利用精心构造的 tar 文件创建这些符号链接，导致指向不安全的位置。
  
- **示例**: 攻击者可以在 tar 文件中链接一个子目录到其父目录，从而使得符号链接最终指向 tar 文件的父目录，绕过程序的安全检查。

## 风险
1. **任意文件写入**: 如果攻击者能够控制 tar 文件，他们可以在系统中任意位置写入文件，权限与运行解压程序的用户相同。
2. **任意文件读取**: 如果攻击者可以读取已解压的文件，他们也能够读取系统上该进程有权限访问的任意文件。
