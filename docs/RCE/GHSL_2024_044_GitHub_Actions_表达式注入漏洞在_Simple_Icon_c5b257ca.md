# GHSL-2024-044: GitHub Actions 表达式注入漏洞在 Simple Icons 中

## 概述
Simple Icons 存在一个 **表达式注入** 漏洞，这使得攻击者能够接管代码库并窃取机密信息。

## 版本信息
- **受影响版本**: Simple Icons v11.9.0

## 漏洞原因
- **工作流触发**: `autoclose-issues.yml` 工作流在创建新问题时被触发。
- **权限设置**: 默认情况下，该工作流使用具有完全写入权限的 GitHub 令牌。这意味着它可以读写代码库。
- **注入点**: 此工作流将由用户控制的问题标题（`${{ github.event.issue.title }}`）注入到执行步骤的脚本中。

## 风险
- **攻击者控制**: 攻击者可以利用此漏洞接管 GitHub Runner，执行自定义命令并更改代码库内容。
- **秘密泄露**: 该漏洞可能导致工作流中机密信息的窃取和代码库的修改。
