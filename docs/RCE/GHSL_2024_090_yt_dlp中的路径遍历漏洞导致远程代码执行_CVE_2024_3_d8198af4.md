# GHSL-2024-090: yt-dlp中的路径遍历漏洞导致远程代码执行 - CVE-2024-38519

## 漏洞概述

- **软件**: yt-dlp
- **版本**: v2024.04.09
- **漏洞类型**: 路径遍历 (Path Traversal)
- **影响平台**: Windows
- **结果**: 允许任意二进制文件覆盖，可能导致远程代码执行 (RCE)

## 技术背景

### yt-dlp功能

- yt-dlp 是一个下载视频和字幕的工具。
- 用户可以通过命令行选项（如 `--write-subs`、`--write-auto-subs`、`--all-subs` 或 `--write-srt`）来指定下载字幕。
- 提供的提取器（extractor）负责处理不同视频网站的下载。

### 字幕扩展的处理

- 一些提取器会提供字幕的下载URL和扩展名。
- yt-dlp 将使用这些扩展名来生成输出文件名。
- 默认情况下生成的字幕文件名格式为: `%(title)s [%(id)s].%(sub_lang).%(sub_format)s`。
- `sub_format`是从提取的URL中获得的字幕扩展名。

### 漏洞详细分析

- yt-dlp **没有验证**字幕扩展名的有效性。
- 在Windows上，如果攻击者使用特制的字幕扩展名，可能会导致路径遍历漏洞。
- 攻击者可以通过精心构造的链接来覆盖系统中的任意文件。

### 影响

- 这种路径遍历漏洞可以导致未授权的文件系统修改。
- 如果被覆盖的文件是可执行文件，可能会导致远程代码执行（RCE）。

## 如何利用

- 除了依赖于视频托管网站的漏洞，攻击者还可以自行构造特定的下载链接来利用该漏洞。
- 例如，攻击者可引导特定的yt-dlp用户下载受影响的视频及其字幕。
