# GHSL-2024-297_GHSL-2024-298：Plenti中的远程代码执行漏洞

## 技术概述

Plenti是一个网站构建工具。其`serve`命令可以创建一个本地服务器，用于查看网站。然而，该服务器存在两个安全漏洞：

1. **任意文件写入漏洞**：在使用Plenti的`/postLocal`接口时，攻击者可以写入任意文件。这可能导致**远程代码执行**，即通过这些文件运行恶意代码。

2. **任意文件删除漏洞**：同样在`/postLocal`接口下，攻击者可以删除任意文件。这可能导致**信息丢失**，即重要数据被删除。

## 版本信息

- 受影响的Plenti版本：v0.7.1
