# GHSL-2021-066: Total.js中的DoS和远程代码执行漏洞

## 简介

- **软件名称**: Total.js
- **最新版本**: 3.4.8（可从npm获取）

## 漏洞概述

- 在Total.js中，使用`utils.set`函数并传入用户控制的值时，会导致代码注入。
- 攻击者可以在Node.js的上下文中执行任意的JavaScript代码。

## 漏洞类型

1. **拒绝服务（DoS）**:
   - 利用漏洞，可以使程序进入无限循环，导致服务不可用。

2. **代码执行（RCE）**:
   - 利用该漏洞，可以在当前工作目录下创建一个名为“GHSL”的文件，执行任意代码。
