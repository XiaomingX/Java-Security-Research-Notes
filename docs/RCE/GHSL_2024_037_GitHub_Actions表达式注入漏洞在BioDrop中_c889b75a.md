# GHSL-2024-037: GitHub Actions表达式注入漏洞在BioDrop中

## 基本信息
- **项目名**: BioDrop
- **版本**: v2.99.18

## 漏洞概述
BioDrop存在一个漏洞，允许攻击者通过**操作表达式注入**来操控代码库的问题（issues）。

## 漏洞详细
1. **触发条件**: 
   - 当新问题被创建或已有问题被修改时，`check-author-issues.yml`工作流会被触发。
  
2. **权限**:
   - 该工作流具有对问题的写入权限。

3. **漏洞机制**:
   - 工作流使用了来自问题的内容（${{ github.event.issue.body }}），并将其注入到脚本中。
   - 这使得攻击者能控制GitHub Runner并执行自定义命令。

4. **影响**:
   - 攻击者可以随意修改问题内容。

## 漏洞发现
- 该漏洞是通过使用CodeQL工具的JavaScript表达式注入查询发现的。
