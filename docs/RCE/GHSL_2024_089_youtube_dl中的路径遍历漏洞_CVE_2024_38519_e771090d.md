# GHSL-2024-089: youtube-dl中的路径遍历漏洞 - CVE-2024-38519

## 概述
在使用youtube-dl下载带字幕的视频时，它没有验证字幕文件的扩展名。这使得Windows用户面临路径遍历的风险，可能导致任意二进制文件被覆盖。

## youtube-dl
- **受影响的版本**：
  - 夜间版本：v2024.05.16
  - 官方最后版本：v2021.12.17

### 功能
youtube-dl不仅可以下载视频文件，还可以下载字幕，用户可以通过以下选项指定：
- `--write-sub`：下载字幕
- `--write-auto-sub`：下载自动生成的字幕
- `--all-subs`：下载所有可用字幕

每个支持的视频网站都有自己的数据提取器。有些提取器提供字幕URL，并从URL路径解析扩展名。其他一些提取器明确提供了字幕的URL和扩展名。

### 安全问题
- **漏洞**：在Windows上，用户可以通过精心构造的字幕扩展名进行路径遍历攻击。
- **影响**：攻击者可以在用户下载视频时，通过恶意字幕扩展名覆盖系统文件。
- **原因**：youtube-dl简单地信任提取器提供的扩展名，构建输出文件名。

### 技术细节
- 默认输出文件模板：`%(title)s-%(id)s.%(ext)s`
- 字幕文件命名格式：`%(title)s-%(id)s.%(sub_lang).%(sub_format)s`
- Linux系统对路径遍历攻击不敏感，而Windows系统可能受到影响。

### 潜在风险
如果攻击者覆盖了可执行文件，可能导致**远程代码执行**。
