# GHSL-2020-109: Codecov中的命令注入漏洞

## 概述

在Codecov的upload方法中发现了一个命令注入漏洞。使用`codecov-node`库的用户可能并未意识到这个问题，这可能导致他们无意中编写含有漏洞的代码。

## 漏洞细节

### 影响版本

- Commit: **eeff4e1**   

### 漏洞证明

1. 首先安装`codecov`。
2. 创建一个包含特定内容的文件。
3. 运行该文件。

运行后，会创建一个名为`exploit`的文件。

### 注意事项

我们知道CVE-2020-7597的存在，但修复措施不彻底。它只屏蔽了`&`字符，而我们的证明概念代码使用了反引号（```）来绕过这个过滤器。

### 潜在风险

如果库的客户使用不受信任的输入调用这个漏洞的方法，可能会导致远程代码执行的风险。

## 安全建议

建议避免使用可能将字符串解释为shell命令的API。例如，使用`child_process.execFile`而不是`child_process.exec`。
