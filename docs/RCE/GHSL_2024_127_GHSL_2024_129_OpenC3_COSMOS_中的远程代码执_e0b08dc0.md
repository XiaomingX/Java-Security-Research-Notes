# GHSL-2024-127_GHSL-2024-129: OpenC3 COSMOS 中的远程代码执行 (RCE) 和跨站脚本 (XSS) 漏洞概述

在 OpenC3 COSMOS（用于控制卫星和测试设备的网络应用）中发现了多个安全漏洞，这些漏洞可能导致远程代码执行（RCE）通过跨站脚本（XSS）攻击。

## 漏洞概述

### 1. 路径遍历漏洞

- **版本**: OpenC3 COSMOS v5.16.2
- **描述**: 通过 `ScreensController#show` 方法，认证用户能够下载任意 `.txt` 文件。漏洞允许用户操作的参数通过不受控制的路径表达式传入，可能造成信息泄露。
- **利用条件**: 用户需要在 URL 中提供合适的参数（`target` 和 `screen`）以获取所需文件。

### 2. 反射型跨站脚本 (XSS)

- **描述**: 在登录页面 `Login.vue` 中，攻击者可以利用恶意的重定向查询参数执行 JavaScript 代码。在用户登录后，攻击者的代码会在用户的浏览器中执行。
- **攻击方式**: 攻击者可能通过钓鱼邮件或恶意网站引导用户点击链接。

### 3. 远程代码执行 (RCE)

- **描述**: 攻击者能够在服务器上创建和执行 JavaScript 代码，进而可能导致远程代码执行。例如，创建一个 Ruby 脚本并在服务器端运行。
- **风险**: 一旦代码执行，攻击者可对服务器进行任意操作。

### 4. 不加密存储密码

- **描述**: OpenC3 COSMOS 将用户密码以明文存储在浏览器的 LocalStorage 中，易受跨站脚本攻击，攻击者可能窃取此密码。
- **后果**: 这可能使攻击者在 COSMOS 实例中获得永久性的访问权限。
