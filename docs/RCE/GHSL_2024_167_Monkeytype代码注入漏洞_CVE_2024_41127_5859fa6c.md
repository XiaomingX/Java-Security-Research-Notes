# GHSL-2024-167: Monkeytype代码注入漏洞 - CVE-2024-41127

## 概述
Monkeytype存在一个安全漏洞，攻击者可以通过代码注入利用该漏洞，从而获得对拉取请求的写入权限。

## 漏洞详细信息
- **漏洞类型**: 代码注入导致的“中毒管道执行”。
- **受影响的文件**: `ci-failure-comment.yml` GitHub工作流程。
- **触发条件**: 当MonkeyCI工作流程完成时，该工作流程会被触发。

## 漏洞机制
1. **工作流程**: `ci-failure-comment.yml` 在CI工作流程完成时执行。
2. **获取数据**: 它会下载由触发的工作流程上传的一个"artifact"。
3. **数据处理**: 将该Artifact中的`./pr_num/pr_num.txt`内容分配给`steps.pr_num_reader.outputs.content`变量。
4. **缺陷**: 该变量没有被验证是否为数字，之后它被插入到一个JavaScript脚本中。
5. **攻击风险**: 攻击者可以改变被执行的代码，从而获得拉取请求的写入权限。
