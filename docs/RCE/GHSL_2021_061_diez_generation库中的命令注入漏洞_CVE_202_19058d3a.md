# GHSL-2021-061: @diez/generation库中的命令注入漏洞 - CVE-2021-32830

## 概述
`@diez/generation`库中的`locateFont`方法存在命令注入漏洞。这意味着使用这个库的开发者可能会无意中写出含有安全漏洞的代码。

## 漏洞详情
- **发现时间**: 2021年3月25日
- **受影响的方法**: `locateFont`
- **操作系统**: 该漏洞在MacOS或经过特定修补的Unix系统上可被攻击利用。

## 漏洞示例
1. 安装`@diez/generation`库。
2. 创建一个包含特定内容的文件。
3. 运行该文件后，会在文件系统中生成一个名为`exploit`的文件。

## 风险
如果该库的用户在调用受影响的方法时传入不可信的输入，攻击者可利用此漏洞进行远程代码执行。
