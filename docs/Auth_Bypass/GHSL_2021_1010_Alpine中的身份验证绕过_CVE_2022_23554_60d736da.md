# GHSL-2021-1010: Alpine中的身份验证绕过 - CVE-2022-23554

## 问题概述

Alpine版本1.10.2存在一个身份验证绕过的漏洞。具体而言，AuthenticationFilter组件无法正确处理某些请求URI，从而可能导致未授权访问。

## 技术细节

- **漏洞类型**: 身份验证绕过
- **影响范围**: 通过特定的URL路径访问swagger接口
- **示例URL**: `/api/foo;%2fapi%2fswagger`

### 漏洞原理

AuthenticationFilter使用请求的URI来判断用户是否在访问swagger接口。当请求的路径格式不正确，例如使用拼接的符号“;%2f”，就会导致AuthenticationFilter无法正确评估身份验证条件。这种情况下，系统将继续处理请求，但不会为用户分配principal对象。

### 影响

虽然此漏洞可能导致非授权的访问，但无法实现用户冒充。
