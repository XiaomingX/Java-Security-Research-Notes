# GHSL-2023-249: Meshery中的SQL注入漏洞 - CVE-2024-29031

Meshery在版本0.6.181及之前存在一个SQL注入漏洞，攻击者可以通过`GetMeshSyncResources`的`order`参数获取敏感信息。

## Meshery概述

Meshery项目在API地址`/api/system/meshsync/resources`处提供了`GetMeshSyncResources`函数。在`meshync_handler.go`文件的第135行，`order`查询参数直接用于构造SQL查询。这可能导致安全问题，如：

- **任意文件写入**：利用SQL注入和堆叠查询，可以执行`ATTACH DATABASE`命令。
- **数据库数据被访问和修改**：攻击者可能访问或修改数据库中存储的任意数据，如性能配置、Meshery应用数据或Kubernetes配置。

## 潜在风险

- 任意数据的读写可能会导致严重问题，这取决于多个因素。
