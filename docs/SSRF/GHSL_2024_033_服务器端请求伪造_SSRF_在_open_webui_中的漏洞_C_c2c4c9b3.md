# GHSL-2024-033: 服务器端请求伪造（SSRF）在 open-webui 中的漏洞 - CVE-2024-30256

## 概述
open-webui 存在一个经过身份验证的盲服务器端请求伪造漏洞。

### 受影响版本
- open-webui v0.1.111

## 漏洞细节
- **漏洞路径**: 下载接口
- **影响**: 用户输入的 URL (通过 GET 参数) 被下载文件流功能使用，导致通过服务器发送 GET 请求，出现盲 SSRF 漏洞。

## 漏洞影响
该漏洞允许攻击者通过 open-webui 服务器发送请求，这可能会：
- 探测其他潜在的服务器漏洞
- 访问服务器内部网络上的其他后端系统
